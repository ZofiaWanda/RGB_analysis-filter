# RGB_analysis-filter
Aplikacja do filtrowania i analizy obrazu z wykorzystaniem przestrzeni kolorów RGB. Umożliwia przetwarzanie zdjęć mikroskopowych (formaty: PNG, JPG, TIFF) poprzez separację kanałów barwnych oraz obliczanie powierzchni obszarów o określonym kolorze. Narzędzie wspiera analizę barwień, pomagając w ilościowej ocenie struktur widocznych na obrazach.

Aplikacja została stworzona w programie MATLAB. 

Jak pobrać aplikację?

1. Przejdź do zakładki Releases po prawej stronie tego repozytorium.
2. Pobierz plik instalacyjny (.exe).
3. Uruchom pobrany plik jako Administrator.
